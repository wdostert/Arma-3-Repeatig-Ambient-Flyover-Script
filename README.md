# Arma-3-Repeatig-Ambient-Flyover-Script

Configurable script for repeating ambient flyovers.  Flyovers occur over the median position of all players.
Plane type, flyover height, flyover radius, direction arc, and interval can all be customized by editing the variables at the start of the function.
To stop the function run the following code server-side:
flyover_enabled = false;
